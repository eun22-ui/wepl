> ### 아키텍쳐 / 기술스택

* 운영체제 : Amazon Linux
* 사용언어 : Web backend / Web frontend


                Python, AWS Lambda, Amazon RDS (MySQL) / HTML5, CSS3, JS, Amazon S3

* 오케스트레이션 : Docker, Amazon ECS
* Server : AWS Linux(c7g.medium)

* CI/CD : AWS CodePipeline, AWS CodeBuild, AWS Lambda
